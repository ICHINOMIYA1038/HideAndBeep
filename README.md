# Rule
・プレイヤーは門を開けて小学校を脱出することが目的です。<br>
・敵のゾンビに触れてしまうとゲームオーバーです。<br>
・各部屋には、インタラクトできるオブジェクトがあります。<br>
・大きな音を出すオブジェクトに触ると、ゾンビを怯ませることができます。<br>
・しかし、音のなるオブジェクトは、ゾンビを引き寄せる原因にもなります。<br>
・うまくゾンビを誘導して、隙を狙って脱出しましょう。<br>
