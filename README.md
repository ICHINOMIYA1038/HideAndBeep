# Rule
・プレイヤーは門を開けて小学校を**脱出すること**が目的です。<br>
・敵のゾンビに触れてしまうと**ゲームオーバー**です。<br>
・各部屋には、**インタラクトできるオブジェクト**があります。<br>
・大きな音を出すオブジェクトに触ると、**ゾンビを怯ませる**ことができます。<br>
・しかし、**音のなるオブジェクトは、ゾンビを引き寄せる原因**にもなります。<br>
・うまくゾンビを誘導して、**隙を狙って脱出**しましょう。<br>
